# Kikoautos · Design System

Design System oficial da Kikoautos. Fonte única para cores, tipografia, componentes, motion e voz do universo digital.

## Rodando localmente

Abra `index.html` diretamente no navegador. Todas as dependências (fontes, SVGs, estilos e scripts) estão embutidas ou referenciadas em `assets/`.

## Estrutura

- `index.html` — documento principal do design system
- `assets/logo-kiko.svg` — logo usado na barra lateral
